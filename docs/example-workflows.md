# Example Workflows

## Onboarding

Input:

```text
Plan onboarding for Sara, full-time employee in Europe, joining Engineering next Monday.
```

Output includes license recommendation, group recommendation, SaaS provisioning payloads, Graph-style planned actions, safety notes, and an audit report.

## Offboarding

Input:

```text
Create an offboarding checklist for Priya leaving on Friday.
```

Output includes sign-in disablement planning, session revocation planning, license removal planning, group removal planning, manager handoff, SaaS deprovisioning payloads, and an audit report.

## Access Troubleshooting

Input:

```text
Maria cannot access Teams.
```

Output checks the mock tenant state for license, group membership, account status, region, conditional access placeholder, and device compliance placeholder.

## Ticket Analysis

Input:

```text
Analyze this ticket: Maria can sign in but Teams says the license is missing.
```

Output classifies category, urgency, likely system, next steps, approvals, automation candidacy, and risk.
