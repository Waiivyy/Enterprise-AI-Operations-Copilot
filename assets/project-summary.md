# Project Summary

Enterprise AI Operations Copilot is a simulation-first technical lab for AI-assisted enterprise IT workflow planning. It combines a deterministic mock LLM provider, an explicit mock tool router, fake Microsoft Graph-style state, advisor modules, ticket analysis, and audit-style report generation.

The default mode demonstrates architecture and control flow rather than advanced live model reasoning. It requires no paid services and no credentials, while the provider interface leaves room for a future OpenAI-compatible implementation behind explicit configuration.

The current demo covers onboarding, offboarding, Teams access troubleshooting, group membership issues, device compliance tickets, and report generation with public-safe sample data.
